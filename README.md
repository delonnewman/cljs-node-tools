NAME
====

cljs-node-tools - A collection of tools for [ClojureScript](https://clojurescript.org) on Node.js.

SYNOPSIS
========

    (use 'js.node.tools)
    
    (spit "test.txt" "Hello")
    (slurp "test.txt") => "Hello"

    (sh "ls -a")
